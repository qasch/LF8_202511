# 15 Übungen zur OOP in Java für Einsteiger

## Übung 1: Klassen und Objekte erstellen
Erstelle eine Klasse `Person`, die die folgenden Attribute enthält:
- `name` (String)
- `alter` (int)

Erstelle ein Objekt der Klasse `Person` und setze die Attribute über einen Konstruktor. Gib den Namen und das Alter der Person in der `main`-Methode aus.

## Übung 2: Getter und Setter Methoden
Erstelle eine Klasse `Buch` mit den Attributen:
- `titel` (String)
- `autor` (String)
- `preis` (double)

Füge Getter- und Setter-Methoden für diese Attribute hinzu und teste sie in der `main`-Methode.

## Übung 3: Konstruktoren verwenden
Erstelle eine Klasse `Auto` mit den Attributen:
- `marke` (String)
- `modell` (String)
- `baujahr` (int)

Verwende einen Konstruktor, um die Werte für diese Attribute zu setzen. Erstelle anschließend ein Objekt der Klasse `Auto` und gib die Informationen über das Auto aus.

## Übung 4: Vererbung - Grundklasse und Unterklasse
Erstelle eine Klasse `Tier` mit der Methode `lautGeben()`, die den Text "Geräusch!" ausgibt. Erstelle dann eine Unterklasse `Hund`, die die Methode `lautGeben()` überschreibt und den Text "Wuff!" ausgibt.

## Übung 5: Methode überschreiben (Override)
Erstelle eine Klasse `Fahrzeug` mit einer Methode `fahre()`, die den Text "Das Fahrzeug fährt." ausgibt. Erstelle dann eine Unterklasse `Fahrrad`, die die Methode `fahre()` überschreibt und den Text "Das Fahrrad fährt." ausgibt.

## Übung 6: Abstrakte Klassen
Erstelle eine abstrakte Klasse `Form`, die eine abstrakte Methode `berechneFlaeche()` enthält. Erstelle dann zwei Unterklassen `Kreis` und `Rechteck`, die die Methode `berechneFlaeche()` implementieren.

## Übung 7: Interfaces implementieren
Erstelle ein Interface `Fliegbar` mit der Methode `fliegen()`. Erstelle dann eine Klasse `Vogel`, die das Interface implementiert und die Methode `fliegen()` mit einer entsprechenden Implementierung versieht.

## Übung 8: Mehrere Interfaces implementieren
Erstelle zwei Interfaces:
- `Schwimmbar` mit der Methode `schwimmen()`
- `Fliegbar` mit der Methode `fliegen()`

Erstelle eine Klasse `Ente`, die beide Interfaces implementiert. Implementiere beide Methoden in der Klasse.

## Übung 9: Polymorphismus
Erstelle eine Klasse `Tier` mit einer Methode `geraeuschMachen()`. Erstelle zwei Unterklassen `Katze` und `Hund`, die beide die Methode `geraeuschMachen()` überschreiben. Zeige, wie Polymorphismus funktioniert, indem du eine Methode verwendest, die `Tier`-Objekte entgegennimmt, aber je nach dem tatsächlichen Objekttyp eine andere Ausgabe erzeugt.

## Übung 10: Konstruktoren und Vererbung
Erstelle eine Klasse `Mitarbeiter` mit den Attributen `name` (String) und `gehalt` (double). Erstelle eine Unterklasse `Manager`, die zusätzlich das Attribut `bonus` (double) enthält. Verwende Konstruktoren, um die Werte zu setzen.

## Übung 11: Statische Methoden
Erstelle eine Klasse `Mathematik`, die eine statische Methode `addiere(int a, int b)` enthält, die die Summe der beiden Zahlen zurückgibt. Teste die Methode in der `main`-Methode.

## Übung 12: Arrays und Objekte
Erstelle eine Klasse `Student` mit den Attributen `name` und `matrikelNummer`. Erstelle ein Array von `Student`-Objekten und initialisiere es mit einigen Studenten. Gib die Informationen jedes Studenten aus.

## Übung 13: Klassenmethoden vs. Instanzmethoden
Erstelle eine Klasse `Taschenrechner`, die eine statische Methode `addiere()` und eine Instanzmethode `subtrahiere()` enthält. Zeige in der `main`-Methode, wie man beide Methoden verwendet.

## Übung 14: Vererbung und Konstruktoren
Erstelle eine abstrakte Klasse `Computer` mit den Attributen `prozessor` und `ram`. Erstelle zwei Unterklassen `Laptop` und `Desktop`, die jeweils einen spezifischen Konstruktor haben, um die Attribute zu initialisieren.

## Übung 15: Exception Handling in OOP
Erstelle eine Klasse `Divisionsrechner`, die eine Methode `teile(int a, int b)` enthält, welche eine Division durchführt. Füge eine Fehlerbehandlung hinzu, die eine Ausnahme wirft, wenn der Divisor `b` gleich null ist.
